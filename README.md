# Flutter Mock Chat API 🚀

A lightweight, DIY Mock REST API built with Node.js and JSON Server. Designed specifically for testing Flutter chat applications with zero cost and unlimited requests.

## 📌 Features
- **Custom Endpoints:** Matches specific chat and suggestion structures.
- **REST Operations:** Supports `GET` for history and `POST` for sending messages.
- **Deployment Ready:** Configured for local development or free hosting on Render/Vercel.

---

## 🛠️ Tech Stack
- **Engine:** [Node.js](https://nodejs.org/)
- **Server:** [JSON Server](https://github.com/typicode/json-server)
- **Hosting:** Render / GitHub

---
